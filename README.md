# Fiscus – Global Economic & News Intelligence Platform

## Overview

Fiscus is a data intelligence platform that analyzes economic indicators alongside global news trends to identify patterns, detect anomalies, and generate explainable financial insights. The system integrates multi-source datasets such as GST collections, macroeconomic indicators, and global news sentiment to transform raw data into structured intelligence through machine learning pipelines and correlation analysis.

The project demonstrates intelligence engineering by focusing on how economic signals and information signals interact rather than treating datasets as isolated analysis tasks.

---

## Key Features

• Multi-dataset integration engine  
Combines economic datasets and global news data for comparative intelligence analysis.

• Economic anomaly detection  
Identifies unusual GST or macroeconomic deviations using statistical and ML techniques.

• News–economy correlation analysis  
Analyzes relationships between news sentiment shifts and economic movement.

• Automated insight generation  
Produces key findings such as major economic changes, sentiment impact signals, and emerging financial patterns.

• Structured ML pipeline  
Clear modular architecture:
Data ingestion → preprocessing → feature engineering → modeling → insight generation → visualization.

• Explainable predictions  
Uses feature importance and trend contribution analysis to interpret model outputs.

• End-to-end data pipeline  
Raw data → cleaning → transformation → modeling → correlation → insight extraction → dashboard.

• Interactive intelligence dashboard  
Visualizes economic trends, sentiment movement, correlations, anomaly alerts, and key insights.
